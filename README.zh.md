## 命令

```shell
# 安装依赖
npm i
# 启动本地预览
ng serve
# 生成正式代码
npm run prod
# 提取待翻译文件
npm run i18n

```

## 改变主题颜色

在 `src/index.html` 添加
```html
<meta name="theme-color" content="#A10000">
```
在 `src/manifest.webmanifest` 设置不一定有效

```
"theme_color": "#A10000",
```
